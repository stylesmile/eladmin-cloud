# sunny-cloud

#### 介绍
基于springcloud 的分布式系统

#### 软件架构

springcloud全家桶
项目中使用的技术如下：

Eureka注册中心 （支持集群部署）
Config注册中心 （支持集群部署）
gateway 服务网关 （支持集群部署）
Hystrix 断路器
Ribbon 负载均衡
Feign 负载均衡
Turbine 聚合监控
Bus 消息总线
